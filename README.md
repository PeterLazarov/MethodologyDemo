# React + NodeJS

## Installation

```bash
cd web
npm install
```

```bash
cd api
npm install
```

## Usage

### API
To run the API for development:
```bash
cd api
npm start
```

To run the tests:
```bash
cd api
npm run test
```

### Web
To run for development:
```bash
cd api
npm start
```
